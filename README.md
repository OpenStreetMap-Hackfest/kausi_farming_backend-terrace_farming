## usage
```
git clone https://github.com/beinganukul/kausi-backend.git
cd kausi-backend
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python3 manage.py migrate
python3 manage.py populate_db
python3 manage.py runserver
```
